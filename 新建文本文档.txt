🎬 交互流程设计

步骤1：用户在左侧操作向量

```javascript
// 用户拖拽向量v
v = [x, y]  // 可实时改变
```

步骤2：矩阵作用于向量

```javascript
// 矩阵A（可编辑）
A = [[a11, a12],
     [a21, a22]]

// 计算 Av
Av = A × v
```

步骤3：右侧显示映射

· 正常情况：显示 Av（可能方向与v不同）
· 特征向量情况：当 Av 与 v 方向相同时，触发动画

步骤4：特征向量动画

```javascript
if (Av 与 v 共线) {
    // 计算特征值 λ
    λ = |Av| / |v| × sign(方向)
    
    // 显示动画：
    1. 虚界背景变为金色
    2. 显示方程式：A·v = λ·v
    3. 特征值λ符号（λ）放大出现
    4. 播放成功音乐
    5. 矩阵锁的齿轮开始转动
}
```
